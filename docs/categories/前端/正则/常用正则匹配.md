# 常用正则匹配

## 邮箱

```js
/(\w+-*\.)*\w+@(\w+-*\w+\.)+[a-zA-Z]/
```

## IP地址

```js
((25[0-5])|(2[0-4]\d)|(1\d{2})(\d{1,2})\.){3}((25[0-5])|(2[0-4]\d)|(1\d{2})(\d{1,2})\.)
```

## URL

```jsx
/^https?:\/\/[\w.-]+(:\d*)?(\/([\w\/-]*(\?\S+)?)?)?/
```

## 日期

```js
(19|20)\d{2}-((0?[1-9])|(10|11|12))-((0?\d)|([12]\d)|(3[01]))
```
