# 读书真的对我有用吗

结论：有，可能不多。

理由：

1. 自上大学以来，工作至今，何曾有读那本书读透过，有那本书对自己影响甚大，以至于影响我这十几年来的人生的？
2. 数学，计算机，文学等书籍，所购甚多，何曾掌握真义，陶冶情操，安身立命。
3. 古文书籍，实体图书并非真爱，无非是读书形式，逼格。若真的爱读书，平板，电脑，手机等电子读书资源丰富，未见如饥似渴。
4. 推己及人，自己读书并不能改变工作，家庭。反而，越来越乱。曾经信书中所言行事，无甚益处。

昔日，上班过程中，说起上学无甚益处，心想上学也只是个底层打工的。其他公事人员反驳说这只是我是这种情况。
人言不差，于他人而言，读书或可改变命运，掌握乾坤。于我而言，读书无甚益处，不过一叶障目不见泰山，反而缩手缩脚，胆小怕事。
