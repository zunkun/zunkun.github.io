# v2rarA安装
系统: Ubuntu 22.04.2 LTS

# 安装
1. 通过脚本安装
```bash
$ curl -Ls https://mirrors.v2raya.org/go.sh | sudo bash
```
2. 启动
```bash
$ sudo systemctl start v2raya.service
```

3. 开机启动
```bash
$ sudo systemctl enable v2raya.service
```
